Features

Everything included in Kegel Morse Hero to support your pelvic floor training.

How does Kegel Morse Hero guide my exercises?

Kegel Morse Hero displays a visual ring animation that expands and contracts to indicate squeeze and relax phases. Each exercise follows a Morse code pattern — dots for short squeezes, dashes for long holds. Haptic feedback vibrates in sync so you can train without looking at the screen.

What Morse code tracks are available?

Kegel Morse Hero includes default tracks: SOS (16 seconds), HELLO (26 seconds), HELP ME (28 seconds), HERO (20 seconds), KEGEL (22 seconds), MORSE (23 seconds), BRAVO (28 seconds), READY (24 seconds), and FOCUS (29 seconds). You can also create unlimited custom tracks with your own Morse code patterns.

How does progress tracking work?

Kegel Morse Hero records every completed session in a calendar view. You can see daily, weekly, and monthly activity at a glance. The history log shows session duration, track used, and speed multiplier for each workout.

What is Zen Mode?

Zen Mode removes all interface elements except the core timing guide. This distraction-free view is designed for focused sessions where you want minimal visual clutter. Activate it from the top-right corner of the main screen.

Can I adjust exercise speed?

Kegel Morse Hero offers 1×, 2×, and 3× speed settings. Beginners can start at 1× for longer, more deliberate movements. Advanced users can increase to 2× or 3× for faster, more challenging sessions.

Does Kegel Morse Hero use signal reminders?

Kegel Morse Hero supports configurable reminders to help you maintain a consistent training schedule. Set daily or custom notifications to stay on track with your pelvic floor routine.

How is my data stored?

All data is stored locally on your device. No internet connection required for exercise sessions. Kegel Morse Hero does not collect personal data, has no ads, and includes no tracking. Optional Google Drive backup and CSV export are available for users who want to save their history externally.
